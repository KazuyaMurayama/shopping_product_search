## 原因分析（Post-mortem：v1 で価格が漏れた原因と対策）

### 症状

v1 レポートで推奨 TOP3 全商品の価格欄が「要最新確認」のまま提出され、さらに中古市場の横断サーチも実施していなかった。

### 根本原因（5 項目）

1. **規約の弱さ（CLAUDE.md）**
   - セクション 3 で価格は「基準」としか書かれておらず、必須項目として扱われていなかった
   - ゲート 2（存在・在庫検証）の合格条件は「レビュー件数 / 価格 / 商品名のいずれか取得」と OR 条件だったため、価格が欠落しても通過できてしまった
2. **スカウトエージェントの雛形**
   - 価格欄に「不明（要最新確認）」が記入されても出力許容だった
   - 価格が取得できない時の再検索ルールが定義されていなかった
3. **report-writer の自己チェック漏れ**
   - 自己チェックリストに「TOP3 全商品に価格数値がある」項目が無く、価格欠落のままレポート確定できてしまった
4. **中古市場サーチの欠落**
   - CLAUDE.md のワークフローに中古再検索フェーズが明示されておらず、各スカウトは自チャネル内の新品検索に閉じていた
   - 「良い商品を特定した後の横断的な中古市場チェック」を担当するエージェントが存在しなかった
5. **オーケストレーター（Claude Code 本体）の運用不備**
   - 各スカウトから「要最新確認」の回答が返ってきた時点で再検索を要請せず、そのまま report-writer に渡してしまった
   - 自己チェック項目が規約側にない以上、オーケストレーター側で push back する動機付けも弱かった

### 対策（コミット [c01d929](https://github.com/KazuyaMurayama/shopping_product_search/commit/c01d929) で構造適用済み）

1. **CLAUDE.md セクション 0** に「失敗パターン 4: 価格未報告 / 価格レンジ未比較」を新規追加。再発したらレポート破棄を明文化
2. **CLAUDE.md セクション 3.1「価格は必須出力項目」** を新設
   - 全候補に価格数値記録を必須化
   - 取得不可なら候補を ⚠️要確認に落として TOP3 除外
   - 価格取得を 2 回試行しても不明なら TOP3 除外と記録
3. **CLAUDE.md セクション 4 ワークフローに「フェーズ 2.5：中古市場サーチ」** を必須化
   - TOP3 確定後に商品名で横断再検索（メルカリ / 楽天中古 / Amazon マーケットプレイス / Amazon Warehouse / Yahoo!ショッピング / Yahoo!オークション）
   - 中古が成立しないカテゴリ（サプリ・医薬品・化粧品・下着等）はスキップして「中古対象外」と明記
4. **CLAUDE.md セクション 6 の自己チェック** に以下を追加
   - 「TOP3 全商品に価格の具体的な数値が入っているか」
   - 「中古対象カテゴリなら中古相場セクションが存在するか」
   - 「取得時刻が記載されているか」
5. **全スカウト**（amazon / rakuten / mercari / iherb / osakado）に価格必須化セクションを追加
   - スニペット取得できない場合の再検索 1 回を義務化
   - それでも不明なら ⚠️ で TOP3 除外
6. **report-writer.md** の自己チェックリスト更新 + 中古相場セクションの書式を明文化
7. **yahoo-shopping-scout サブエージェント** を新規追加
   - Yahoo!ショッピング（store.shopping.yahoo.co.jp）
   - Yahoo!オークション（page.auctions.yahoo.co.jp）
   - 中古市場サーチで頻用されるため独立スカウト化
8. **オーケストレーター運用ルール**: スカウトが「要最新確認」を返したら 1 回は必ず再検索を要請する（本 v2 で実践済み）

## 補足情報

- 価格・在庫は新品検索が **2026-04-08 10:46**、中古市場サーチが **2026-04-08 11:00 頃** のものです。変動する可能性があります
- Amazon の [MAMORIO CARD ¥3,410](https://www.amazon.co.jp/dp/B08LL17T6D) は極めて競争力のある価格で、楽天公式ストアの発売時定価 ¥6,578 税込の約半額です
- 楽天の [MAMORIO CARD](https://item.rakuten.co.jp/mamorio/mamd-001-bk/) / [Chipolo CARD](https://item.rakuten.co.jp/sincere-watch/chipolocard/) 現行価格は 404 応答のため直接取得できませんでした。購入前に楽天ページで直接確認してください
- Tile Slim Gen3（2mm 厚）は販売終了または売り切れが多く、新規購入は [Tile Slim 2022（2.4mm）](https://www.amazon.co.jp/dp/B09HTW9MVK) が現実的です
- 中古市場は 4 チャネル横断サーチを実施しましたが、3 商品とも実勢ほぼ成立せず、**新品購入が現実的** です
- メルカリは Tile スターターパック（Mate+Slim セット）以外の個人出品が極めて少なく、Tile Slim 単品の中古入手は困難です
- メルカリの Tile / MAMORIO / Chipolo は旧所有者からの譲渡解除手続きが必要なケースがあります。購入前にコメントで要確認
- Yahoo!オークションは個別出品 URL の一次特定ができなかったため本レポートから除外しています。ブラウザで直接検索すれば個別出品が見つかる可能性があります
- Amazon の中古出品（マーケットプレイス）と Amazon Warehouse は robots.txt によりスニペットから情報取得できません。個別商品ページで「新品＆中古」タブを直接確認してください
- iHerb / オオサカ堂は本検索対象外（Bluetooth スマートタグの取扱無し）
- **最終確認はユーザー自身で行ってください**
