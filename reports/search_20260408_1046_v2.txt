-- Apple AirTag との比較考察 --

Apple AirTag の基本情報:
  - 形状: コイン型、直径約 31.9mm、厚さ約 8mm
  - 方式: Bluetooth + Apple 探すネットワーク
  - 対応 OS: iOS 専用（Android 非対応）
  - 特徴: U1 超広帯域チップ搭載で正確方向探知（Precision Finding）が可能
  - 価格: Apple 公式 4,780 円 / 4 個パック 15,800 円（参考）

推奨 TOP3 が AirTag より優位な点:
  - 薄さ: AirTag 約 8mm に対し、MAMORIO 1.7mm / Tile Slim 2.4mm / Chipolo 2.4mm
    財布収納時の厚みの実感差は極めて大きい
  - Android 対応: AirTag は iOS 専用、TOP3 は全て iOS/Android 両対応
  - 価格:
    - MAMORIO CARD ¥3,410 < AirTag ¥4,780（約 29% 安）
    - Tile Slim ¥4,719 ≒ AirTag ¥4,780
    - Chipolo CARD ¥5,280 > AirTag ¥4,780
  - 形状: カード型で財布に自然収納、コイン型の AirTag は財布収納に不向き
  - MAMORIO は Qi 充電対応で電池交換不要、紛失時補償付き

AirTag が TOP3 より優位な点:
  - 探索ネットワーク規模: iPhone ユーザー数が莫大で広域トラッキング範囲が広い
    MAMORIO / Tile / Chipolo は独自ネットワークで、ユーザー数は AirTag より小規模
  - U1 チップによる正確方向探知（Precision Finding）機能
  - Apple 純正エコシステムとの統合度（iCloud、ファミリー共有等）
  - 4 個パックがある（複数個を運用しやすい）

使い分けの指針:
  - iPhone 単独運用、屋内精密探索重視 → AirTag
  - Android 併用、または薄さ重視 → MAMORIO / Tile / Chipolo
  - 国内サポート・紛失補償重視、とにかく薄さと価格 → MAMORIO CARD（本推奨）
  - グローバル探索網重視 → Tile Slim
  - OS ロックインを避けたい → Chipolo CARD

-- 原因分析（Post-mortem：v1 で価格が漏れた原因と対策） --

症状:
  v1 レポートで推奨 TOP3 全商品の価格欄が「要最新確認」のまま提出され、
  さらに中古市場の横断サーチも実施していなかった。

根本原因（全 5 項目）:

  1. 規約の弱さ（CLAUDE.md）
     - セクション 3 で価格は「基準」としか書かれておらず、必須項目
       として扱われていなかった
     - ゲート 2（存在・在庫検証）の合格条件は「レビュー件数 / 価格 /
       商品名のいずれか取得」と OR 条件だったため、価格が欠落しても
       通過できてしまった

  2. スカウトエージェントの雛形
     - 価格欄に「不明（要最新確認）」が記入されても出力許容だった
     - 価格が取得できない時の再検索ルールが定義されていなかった

  3. report-writer の自己チェック漏れ
     - 自己チェックリストに「TOP3 全商品に価格数値がある」項目が無く、
       価格欠落のままレポート確定できてしまった

  4. 中古市場サーチの欠落
     - CLAUDE.md のワークフローに中古再検索フェーズが明示されておらず、
       各スカウトは自チャネル内の新品検索に閉じていた
     - 「良い商品を特定した後の横断的な中古市場チェック」を担当する
       エージェントが存在しなかった

  5. オーケストレーター（Claude Code 本体）の運用不備
     - 各スカウトから「要最新確認」の回答が返ってきた時点で再検索を
       要請せず、そのまま report-writer に渡してしまった
