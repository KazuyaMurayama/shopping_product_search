# docs/used-market-rules.md — 中古市場サーチ詳細ルール

CLAUDE.md セクション 4 のフェーズ 2.5（中古市場サーチ）から抽出した
詳細ルールです。中古成立カテゴリかどうかを判定する時だけ参照します。

---

## 中古市場サーチを実施するチャネル

TOP3 候補確定後、以下 4 チャネルを横断再検索する：

- メルカリ（個人間中古）
- 楽天市場内の中古出品
- Amazon マーケットプレイス中古 / Amazon Warehouse
- Yahoo!ショッピング / Yahoo!オークション

各チャネルで価格レンジと状態を取得し、レポートに「中古相場」セクションで
掲載する。

---

## 中古が成立しない / 推奨しないカテゴリ

以下のカテゴリはフェーズ 2.5 をスキップし、レポートに「中古対象外」と明記：

- サプリ・ビタミン・プロテイン等の口に入れるもの
- 医薬品・医療機器
- 化粧品・衛生用品
- 下着・水着
- 食品全般
- 開封済みで価値が落ちる消耗品

---

## 中古購入時の追加注意事項

- 電子機器（スマートタグ、スマートウォッチ等）は電池残量を必ず確認
- アカウント紐付け型（Tile / MAMORIO / Chipolo / AirTag 等）は
  旧所有者からの譲渡解除手続きの要否を購入前に確認
- メルカリは個人出品のため流動性が高い。価格・在庫は短時間で変動
- Yahoo!オークションは終了済み出品が多いため、残り時間と入札状況を確認

---

## 「中古相場」セクションの書式

`reports/TEMPLATE.txt` および `.claude/agents/report-writer.md` を参照。
最低限含める項目：

- 各チャネルの価格レンジ（¥X,XXX 〜 ¥Y,YYY）
- 状態の分布（新品未使用 / 目立った傷なし / 等）
- サンプル URL（個別商品ページのみ）
- 新品 vs 中古の価格差サマリ
- 新品 / 中古のどちらを推奨するか（理由付き）
